# 🧠 Multi-Disease Medical Diagnosis (Deep Learning)

An intelligent deep learning-based system for detecting multiple diseases from medical images such as X-rays, MRI, CT scans, and skin images.

---

## 🚀 Overview

This project uses **Convolutional Neural Networks (CNNs)** to classify diseases across different organs like brain, lung, bone, skin, and eye.
It also compares results with traditional models like **Decision Tree** and **Random Forest**.

---

## 🎯 Features

* Multi-disease prediction
* Image-based diagnosis
* Real-time results
* Django web interface

---

## 🛠️ Tech Stack

* Python, Django
* TensorFlow / Keras (CNN)
* HTML, CSS
* SQLite

---

## ▶️ Run the Project

```bash
git clone https://github.com/PrekshaChidarla/Multi-Disease-Prediction.git
cd Multi-Disease-Prediction
python manage.py runserver
```

Open: http://127.0.0.1:8000/
